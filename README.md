# SOC Investigations

A collection of hands-on Security Operations Center (SOC) investigations performed in simulated lab environments. This repository showcases practical blue team skills through the analysis of security events, alerts, and attack scenarios.

The investigations documented in this repository focus on identifying malicious activity, validating alerts, collecting evidence, and producing structured incident reports.

---

## Objectives

* Perform alert triage and investigation.
* Analyze logs from multiple data sources.
* Identify Indicators of Compromise (IOCs).
* Map adversary behavior to the MITRE ATT&CK framework.
* Document findings using industry-standard investigation methodologies.
* Strengthen incident detection and response skills.

## Investigation Methodology

Each investigation follows a structured workflow:

1. Review and triage the alert.
2. Collect and validate relevant evidence.
3. Analyze logs, artifacts, and indicators.
4. Identify suspicious or malicious activity.
5. Determine the scope and impact.
6. Classify the alert (True Positive / False Positive).
7. Map observed behavior to the MITRE ATT&CK framework.
8. Document findings and provide recommendations.
